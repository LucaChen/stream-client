# alexa-doorman-streaming-client

## Description
This client needs to be installed on the device that will capture images. 

## Dependencies
1. Python 3.6
1. OpenCV 3
1. Flask


## Usage
0. Run `sudo modprobe bcm2835-v4l2` to enable pi camera to work with opencv video capture


## Tested Platforms
1. Windows
1. Raspbian Jessie
